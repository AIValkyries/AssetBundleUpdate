# AssetBundleUpdate
 UnityAssetBundle打包系统，支持多版本下载，增量更新，边玩边下载，分包以及 proto配置文件系统.
 不知如何表达，有兴趣自己研究下有问题 969929242@qq.com
# 主要功能
  * 1.proto excel配置表系统
  * 2.支持单个文件打包以及文件夹打包；
  * 3.支持分包(excel表中进行分包)
  * 4.支持边玩边下载
  * 5.支持断点续传
# 第三方库
* proto序列化excel:https://blog.csdn.net/linshuhe1/article/details/68060544
* zip压缩库
